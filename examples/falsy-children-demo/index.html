Falsy Children Demo - Tacit-DOM

This demo shows how falsy values are automatically filtered out when rendering DOM elements.

Basic Falsy Value Filtering

Code:

div('Hello', null, undefined, false, 0, '', 'World')

Result:

Only "Hello" and "World" are rendered. All falsy values are filtered out.

Conditional Rendering

Code:

div('Status:', showExtra && 'Extra info', showDetails ? 'Details' : null, 'Complete')

Result:

Click the buttons to see how conditional content is dynamically filtered.

Reactive Conditional Rendering

Code:

div('Count:', count || 'No count yet', count > 5 && ' (High!)')

Result:

The count and conditional messages update reactively as signals change.

Complex Conditional Structure

Code:

div( 'User:', user.name, user.isAdmin && ' (Admin)', user.lastLogin && `Last login: ${user.lastLogin}`, !user.isActive && ' (Inactive)' )

Result:

Complex conditional rendering with multiple falsy checks.
