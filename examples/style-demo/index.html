CSS Custom Properties (Variables)

This demo uses CSS custom properties to create a comprehensive theming system. The theme automatically applies to all elements and can be toggled between light and dark modes.

--bg-primary Background color for main content areas
--text-primary Primary text color
--accent-primary Primary accent color for highlights
--border-primary Primary border color
